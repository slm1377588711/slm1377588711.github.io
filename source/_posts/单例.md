# 单例模式
自始至终保证一个对象被创建
1. 构造方法私有化
1. 提供共有静态成员方法实例化对象
1. 私有静态本类类型的变量

<!-- more -->

### 懒汉式(后创建对象)

```
public class User
{
//私有静态本类类型的变量
private static User user;
//构造方法私有化
private User(){}
//共有静态成员方法实例化对象
public static User newInstance(){
if(user==null){
 user = new User();
}
return user;
}
}
```

### 饿汉式(先创建对象)

```
public class User
{
//私有静态本类类型的变量
private static User user = new User();
//构造方法私有化
private User(){}
//共有静态成员方法实例化对象
public static User newInstance(){
return user;
}
}
```
